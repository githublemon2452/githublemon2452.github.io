拍摄图

星图

随笔

第一次爬上来办公室楼顶，天呀，原来这里有这么好的天台，来了两年才发现！（哭）瞬间就成为了中大里新的观星好地方，我不会再去档案馆前面那个草坪了，这里方便多了！

不过上面没有什么座椅和可以依靠的地方，手机随便靠着泡沫花盆拍了一张，回头分析了一下，找到了，是北冕座哈。

以后我会常上来哈！

  • 使用设备

拍摄图

星图

随笔

第一次用上自己的双筒望远镜来拍，为此专门准备了三脚架和望远镜转接器。但是装备还是过于粗糙了，稳定性不是十分好，调整也花了很长时间，特别是手机连接望远镜的地方，太不稳了，还好最后凑合搞出来了，搞得满头大汗呀。

拍了蛮多张的的，有拍到卫星了，感觉任务完成了，应该还可以调节拍摄参数拍得好一些，但是有点晚也有点累了，于是就收工了。

嗯，还是要补点拍摄的理论知识呀。

参数

  • 时间
    2019/7/13 20:30-22:00
  • 地点
    广州海心沙亚运公园
  • 使用设备
    很多高端设备哈，不懂
  • 拍摄参数

拍摄图

  • 月球

  • 木星及其4个卫星

  • 月球

今天胡睿去海心沙亚运公园那边溜达，发现了广州天文协会的活动，有好多高大上的仪器和好多大神，诶，于是我立马过去耍了，哈哈，这算是找到组织了呢。

用他们的仪器看了月亮和火星，哈哈，感觉自己平时用双筒看的像坨屎了！月亮很清晰，火星纹理也看到，虽然没有想象中的大哈。

2019/7/17 满月 土星环

参数

  • 时间
    2019/7/17 19:30-23:00
  • 地点
    中山大学光电楼楼顶
  • 使用设备
    redmi + 博冠猎手II 10x50双筒望远镜/Boto天文望远镜 + 三脚架
  • 拍摄参数
    偷懒不写啦

拍摄图

  • 满月



  • 土星环

星图

  • 月球
  • 土星环

随笔

今天是满月，早有计划想看一波，没想到天气还不错，傍晚天还没有黑就来到楼顶了，满月真的好美好美的！傍晚还有有点泛红的样子。在双筒中也看得很清楚。于是拍拍拍，原来之前手机调的焦距还可以更小，这样拍出来就更大哈。

今晚还是大佬柴建的天文望远镜，原本我对这个镜并没有什么期待，没想到这个镜还行啊，看看月亮，看看木星，最后我还调出了土星！哈，第一次见土星环呀，好开心！

诶，今夜月色真美，只是…
